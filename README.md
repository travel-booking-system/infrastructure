# Infrastructure Repository

This repository contains the shared infrastructure used by the travel booking platform.

## Purpose
- Local development setup
- Shared environment configuration
- Foundation for CI/CD and deployment

## What is included
- Docker Compose for local development
- Environment variable templates
- CI/CD documentation (high level)

## What is NOT included (yet)
- Cloud infrastructure
- Kubernetes
- Production deployment

These will be added incrementally as the project evolves.
